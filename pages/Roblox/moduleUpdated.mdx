# `moduleUpdated( callback )`

Fires event when a newer configuration module is received from the Roblox website.


## Parameters

- `callback` (function): Function to be called upon a module update. New config value and old config value are passed to callback by default. 


## Usage
```lua filename="moduleUpdated"
local connection = configs.moduleUpdated(function()
  print("Updated")
end)
```
