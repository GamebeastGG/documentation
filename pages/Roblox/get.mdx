# `get( config )`

This function is used to retrieve the value of one or more specific configs. It can accept either a single string or a list of strings as the `config` parameter.


## Parameters

- `config` (string or list of strings): The name of the config(s) to retrieve. You can provide either a single string or a list of strings.


## Usage
```lua
local sunPosition = configs.get("SunPosition")
```

```lua 
local sunPosition = configs.get({"SunPosition"})
```

```lua 
configs.get("SunPosition"), configs.get({"NPCs", "Spider", "Health"}
```
