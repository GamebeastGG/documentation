# `Get( config )`

This function is used to retrieve the value of a specific config. It can accept either a single string or a list of strings as the `config` parameter.


## Parameters

- `config` (string or list of strings): The name of the config to retrieve. You can provide either a single string or a list of strings.


## Usage
```lua
local sunPosition = configs:Get("SunPosition")
```

```lua 
local sunPosition = configs:Get({"SunPosition"})
```

```lua 
local health = configs:Get({"NPCs", "Spider", "Health"})
```
