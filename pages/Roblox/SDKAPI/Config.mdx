# Config
Context:
`server` `client`

This API is used to retrieve the values of Gamebeast configs.

```lua
local Gamebeast = require(game:GetService("ReplicatedStorage"):WaitForChild("Gamebeast"))
local GamebeastConfig = Gamebeast:GetService("Config")
```

## `:Get(path)`
This function is used to retrieve the value of a specific config. It can accept either a single string or a list of strings as the `config` parameter. 

### Parameters

- `path : string | {string}`: The name of the config to retrieve. You can provide either a single string or a list of strings.

**Returns:** `{[string] : any} | any`

### Usage
```lua
-- string
local SunPosition = GamebeastConfig:Get("SunPosition")

-- Path (array of strings)
local SunPosition = GamebeastConfig:Get({"SunPosition"})
local Health = GamebeastConfig:Get({"NPCs", "Spider", "Health"})
```

## `:IsReady()`
Returns a boolean indicating whether the config are loaded and ready to be used.

### Parameters

- N/A

**Returns:** `boolean`

### Usage
```lua
if GamebeastConfig:IsReady() then
    print("Configs are ready!")
end
```


## `:OnChanged(path, callback)`
This function is used to retrieve the value of a specific config. It can accept either a single string or a list of strings as the `config` parameter. 

### Parameters

- `path : string | {string}`: The name of the config to listen to. You can provide either a single string or a list of strings.
- `callback : (newValue : any, oldValue : any) -> ()`: Callback function fired when the config on the provided path changes.

**Returns:** `RBXScriptSignal`

### Usage
```lua
-- string
GamebeastConfig:OnChanged("SunPosition", function(newValue, oldValue)
    print("New config", newValue)
end)

-- Path (array of strings)
local ChangedConnection = GamebeastConfig:OnChanged({"Lighting", "Atmosphere"}, function(newValue, oldValue)
    print("New config", newValue)
end)
```
