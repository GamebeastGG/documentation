# Usage

## Calling into Gamebeast from Roblox

```lua filename="example.lua" {3} copy
local replicatedStorage = game:GetService("ReplicatedStorage")
local configs = require(replicatedStorage.Gamebeast)

local health = configs.get({"NPCs", "Spider", "Health"})
print(health)

configs.onChanged({"NPCs", "Spider", "Health"}, function(newVal)
  health = newVal
end)
```
