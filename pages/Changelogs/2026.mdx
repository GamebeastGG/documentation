# Platform Changelog 2026

## Release v26.1.1a (2026-01-08)

**New Features**

- Added ability to edit heatmap names and descriptions in data management page
- Implemented alert banner system for displaying notifications
- Added link buttons on marker types to navigate to usage page in data management page

**Improvements**

- Limited insight queries to 15 metrics maximum and funnel queries to 32 metrics maximum
- Moved experiment "Add Filter" button below filter list and swapped order of user/server experiment modes
- Increased project selection popover height and improved project name visibility
- Removed Studio tag from deeplinks table since deeplinks are not scoped to environment

**Bug Fixes**

- Fixed sign-in issues on Safari for localhost environments
- Fixed non-functional trash can button for experiment date ranges
- Fixed bug preventing custom unit decimals from being set to 0
- Fixed unintended toggle behavior where clicking setting names would change toggle states for brand opportunities and ad settings
